# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-04-05

### Added
- Initial release
- QR code generation for USD and KHR
- Static and dynamic QR codes
- MD5 hash generation
- Bakong API integration (deep links, payment checking)
- QR code image generation (PNG, JPEG, WebP, SVG, base64)
- Full TypeScript support
